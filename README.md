## ▪ About

I'm **Zenda** (Nguyen Trung Nam) — a **Backend Developer** with a Software Engineering degree from **FPT University** and hands-on experience across the **.NET ecosystem** (.NET Core, ASP.NET Core, Entity Framework Core).

I focus on architecting **scalable RESTful APIs**, designing optimized relational databases with **SQL Server**, and applying **Clean Architecture** principles. I've also built **blockchain-integrated** systems — anchoring academic records on a private **Ethereum Quorum** ledger for tamper-proof verification.

I enjoy working in **Agile/Scrum** teams: reviewing code, troubleshooting complex defects, and shipping high-performance server-side solutions.

**Open to** — Backend / Full-stack developer roles · onchain & product engineering collaborations
